# Vowels

**Count [Vowels](https://en.wikipedia.org/wiki/Vowel)**

## Table of Contents

1. [Run](#run)
2. [Options](#options)
3. [Examples](#examples)

## Run

```bash
npm install
```

```bash
node index.js [options]
```

or

```bash
npm run start -- [options]
```

### Options

- **-s**, **--str**

    <pre>[<em><b>required</b></em>] String for count vowels</pre>

### Examples

```bash
$ node index.js -s 'Hello World!'
a: 0
e: 1
i: 0
o: 2
u: 0
Execution time (hr): 0s 0.062561ms
```
