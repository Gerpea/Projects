**Reverse a String**

## Table of Contents

## Run

```bash
npm install
```

```bash
node index.js [options]
```

or

```bash
npm run start -- [options]
```

### Options

- **-s**, **--str**

    <pre>[<em><b>required</b></em>] String for reverse</pre>

$ node index.js -s 'Hello World!'
