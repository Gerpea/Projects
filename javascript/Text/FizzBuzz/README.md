## Run

```bash
npm install
```

```bash
node index.js [options]
```

or

```bash
npm run start -- [options]
```

### Options

- **-n**, **--number**

    <pre>[<em><b>optional</b></em>] Number to fizzbuzz
    <em>default is</em> <b>100</b></pre>
