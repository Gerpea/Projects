# Distances

**Calculate Distance Between Two Cities with [Haversine formula](https://en.wikipedia.org/wiki/Haversine_formula)**

## Table of Contents

1. [Run](#run)
2. [Example](#examples)

## Run

```bash
npm install
```

```bash
node index.js
```

or

```bash
npm run start
```
