# Complex Number

**Complex Number Algebra**

## Table of Contents

1. [Run](#run)
2. [Example](#examples)

## Run

```bash
npm install
```

```bash
npm run build
```

```bash
npm run start
```

### Examples

```bash
$ npm run start

> complexnumber@0.0.1 start
> node dist/index.js

a = 1 + 2i
b = 1 + 3i

a + b = 2 + 5i
a - b = 0 - 1i

a * b = -5 + 5i
a / b = 0.7 - 0.1i

-a = -1 - 2i
-b = -1 - 3i

1/a = 0.2 - 0.4i
1/b = 0.1 - 0.3i
```
