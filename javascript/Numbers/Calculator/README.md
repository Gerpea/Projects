# Calculator

**Basic Calculator**

## Table of Contents

1. [Run](#run)
2. [Description](#description)
3. [Examples](#examples)

## Run

```bash
npm install
```

```bash
npm run start
```

### Description

This is a basic calculator

It supports the following operators:

- '+'
- '-'
- '\*'
- '/'

Also you can you parethesis for defining order

    (1 + 2) * 3 = 9

### Examples

```bash
$ npm run start

> calculator@0.0.1 start
> babel-node index.js

> 1 + 1
2
> 2*2
4
> 4/3
1.33333333333333333333
> 2-1
1
> --1
1
> +-1-2+3
0
> (1+4)*2/(4/((3+1)-1*0.44))
8.9
```
