# Unit Converter

**Unit Converter**

## Table of Contents

1. [Run](#run)

## Run

```bash
npm install
```

```bash
npm run start
```

or

```bash
npm run build
node dist/index.js
```

### Examples

```bash
$ npm start

> unitconverter@0.0.1 start
> ts-node src/index.ts

? Enter unit to convert from mg
? Enter unit to convert to g
? Enter value 100
100 mg = 0.1 g
```
